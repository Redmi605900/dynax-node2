# 🚀 DYNAX - Layer 1 Blockchain

**DYNAX** is a fully decentralized Layer 1 blockchain with native smart contract support (DVM), built from the ground up without any intermediaries.

## 🌟 Features

### Core Blockchain
- ✅ **Proof-of-Work Consensus** - SHA3-256 hashing algorithm
- ✅ **Decentralized Network** - Peer-to-peer architecture
- ✅ **726+ Blocks** - Active blockchain with 740+ transactions
- ✅ **343,032 DYX** - Total supply mined

### Smart Contracts (DVM)
- ✅ **Contract Deployment** - Deploy custom smart contracts
- ✅ **Contract Execution** - Execute contract methods (get, set, balance, transfer)
- ✅ **Persistent Storage** - State management for contracts
- ✅ **Events System** - Track contract events and state changes
- ✅ **ERC20 Template** - Standard token implementation

### Web Platform
- 🏠 **Landing Page** - https://dynax-website-2.onrender.com/
- 📊 **Dashboard** - Network statistics and monitoring
- 🔍 **Block Explorer** - Browse blocks, transactions, and contracts
- 💱 **DEX** - Decentralized exchange with AMM (x*y=k)
- 🔧 **DVM Interface** - Deploy and execute smart contracts
- 📜 **Events Explorer** - Real-time event tracking
- 📄 **Whitepaper** - Technical documentation

## 📡 API Endpoints

### Node API
- `GET /` - Node status
- `GET /chain` - Full blockchain
- `GET /blocks/last` - Latest block
- `GET /dvm/contracts` - List all contracts
- `POST /dvm/deploy` - Deploy new contract
- `POST /dvm/execute` - Execute contract method
- `GET /events/logs` - Event logs
- `GET /events/stats` - Event statistics
